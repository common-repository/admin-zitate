== Description ==

Ein Plugin, welches beliebte Zitate direkt im Dashboard anzeigt.
Das Ganze ist in Deutsch und wird in Zukunft um Zitate und Sprachen erweitert.
Die Zitate können in einer Textdatei im Plugin-Ordner angepasst werden.


Bei Fragen, Anregungen oder Zitat-Vorschlägen: schuhknecht@design-insel.de

webseite: [www.design-insel.de](http://www.design-insel.de "Design Insel Webdesign")

== Installation ==

1. Kopieren Sie den Plugin-Ordner in "/wp-content/plugins/".
2. Das Plugin in der Wordpress-Administration unter "Plugins" aktivieren.

== Frequently Asked Questions ==

=Kann ich Zitate hinzufügen?=

Ja, im Plugin-ordner gibt es eine Datei namens "zitate.txt" dort können die Zitate angepasst werden. Die Zitate müssen zeilenweiße angeordnet sein.
